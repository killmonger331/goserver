# Go Server

## Motivation
This project exists to build and run a simple Go HTTP server locally as a foundation for future containerization and deployment.

## Quick Start
```bash
git clone https://github.com/killmonger331/goserver.git
cd goserver
go run .
```
# Usage
Once rnning, the server listens on a local port and serves HTTP responses.
